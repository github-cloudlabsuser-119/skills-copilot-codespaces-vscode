## Create a Virtual Network in Azure Portal

This code snippet demonstrates how to create a virtual network in the Azure portal.

### Prerequisites
- Azure subscription
- Azure portal access

### Steps
1. Log in to the Azure portal.
2. Navigate to the Azure portal dashboard.
3. Click on the **Create a resource** button.
4. In the search bar, type **Virtual Network** and select **Virtual Network** from the search results.
5. Click on the **Create** button to start the virtual network creation process.
6. Fill in the required details such as **Name**, **Address space**, and **Subnet**.
7. Click on the **Review + Create** button to review the settings.
8. After reviewing, click on the **Create** button to create the virtual network.
9. Wait for the deployment to complete.
10. Once the deployment is successful, you can access and manage your virtual network in the Azure portal.
